## 📝 Technical Details

### Color Matching Algorithm:
- Uses **Euclidean distance** in RGB color space
- Formula: `√((R₁-R₂)² + (G₁-G₂)² + (B₁-B₂)²)`
- Maximum possible distance: 441.67
- All matches under distance of 50
- Similarity = `100 - (distance / 441.67 * 100)`
